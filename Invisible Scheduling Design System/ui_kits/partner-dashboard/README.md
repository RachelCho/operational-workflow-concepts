# Invisible Scheduling Partner Dashboard — UI Kit

A pixel-fidelity recreation of the Invisible Scheduling Partner Dashboard, built from the Figma design system (`Design System v1.fig`).

## Screens
- **Today** — overview with KPI widgets + visit queue table
- **Visits** — filterable list of scheduled visits
- **Clinicians** — roster with scoring indicators

## Components
- `Sidebar.jsx` — left nav with product brand
- `TopBar.jsx` — page title + search + user avatar
- `WidgetCard.jsx` — KPI card (label + value + delta)
- `DashboardTable.jsx` — dense row layout with status chips
- `StatusChip.jsx` — status pills (Completed / Needs Assignment / In Review / Failed)
- `Button.jsx` — primary / secondary / ghost
- `Alert.jsx` — inline alerts
- `DatePicker.jsx` — mini calendar

See `index.html` for a click-thru prototype.
